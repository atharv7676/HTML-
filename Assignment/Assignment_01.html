Mathematical Formulas:


1. (a + b)2 = a2 + 2ab + b2
2. (a − b)2 = a2 − 2ab + b2
3. a2 − b2 = (a + b)(a − b)
4. (a + b)3 = a3 + 3a2b + 3ab2 + b3
5. (a − b)3 = a3 − 3a2b + 3ab2 − b3
6. a3 + b3 = (a + b)(a2 − ab + b2)
7. a3 − b3 = (a − b)(a2 + ab + b2)
8. (a + b + c)2 = a2 + b2 + c2 + 2(ab + bc + ca)
9. (x + y)2 = x2 + y2 + 2xy
10. (m - n)2 = m2 + n2 − 2mn
